# 40-LAB

Experiments, RFC proposals, and audit reports.

- **Breakdowns** — `BREAKDOWN-{slug}.md`, authored from [`50-TEMPLATES/TEMPLATE-breakdown.md`](../50-TEMPLATES/TEMPLATE-breakdown.md). The uncompressed research report a card is built from — the source of truth, kept as genealogy. See [`10-GOVERNANCE/PIPELINE-card-assembly.md`](../10-GOVERNANCE/PIPELINE-card-assembly.md).
- **RFCs** — `RFC-{NNNN}-{slug}--proposed.md`, authored from [`50-TEMPLATES/TEMPLATE-rfc.md`](../50-TEMPLATES/TEMPLATE-rfc.md). Proposals for new blocks, principle changes, or process changes.
- **Audits** — `AUDIT-{date}--draft.md`. Drift reports grading cards against PRINCIPLES + recent stewards' log entries.
- **Experimental blocks** — trial blocks live here until promoted to `20-BLOCKS/` (see ADR-0002). Trial budget: ≥3 real Supercards must use it; sunsets at 90 days if not promoted.
